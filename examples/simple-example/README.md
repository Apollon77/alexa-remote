# Simple example

This is the simplest app you can have that communicates with Alexa.  
This example will connect with Alexa, give you the proxy url to authenticate and list the devices.  
You will need to enter user and password every time you run this. It is possible to save the cookie and the macDms
so you don't need to login again. See coments or other examples on how to do that.  

To run this example, execute:
`node example.js`
